## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/ChandraVerse/chandraverse-portfolio.git
cd chandraverse-portfolio

# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## 🌐 Deploy

### Vercel (Recommended)
1. Push to GitHub
2. Connect repo at [vercel.com](https://vercel.com)
3. Framework: **Vite** — Deploy!

### Netlify
1. Push to GitHub
2. Connect repo at [netlify.com](https://netlify.com)
3. Build command: `npm run build`, Publish dir: `dist`
